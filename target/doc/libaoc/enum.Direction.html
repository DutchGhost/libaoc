Enum libaoc::Direction [−] [src]

pub enum Direction {
    Up,
    Down,
    Right,
    Left,
    Init,
}

An enum to represent a direction. Is great to use in maps, or when 'following' some kind of line. Also supports an init variant. When the current variant is init, using turn_right returns the variant right. When the current variant is init, using turn_left returns the variant left.
